# 学籍番号バリデーション＆集計コマンド

## 概要
このコマンドは標準入力から与えられた複数の学籍番号を検証し、その結果を標準出力に一覧表示するとともに、判定結果の集計を行うプログラムです。学籍番号の形式誤りや無効な値をエラーコードとして分類することで、入力データの確認や事前チェック用途に利用できます。

## 機能概要
- 標準入力から複数行の学籍番号を受け取って処理します。
- 学籍番号を以下の形式で検証します。  
  - 年（2桁）
  - 学科コード（3桁）
  - 個別番号（2桁）
- 各行ごとにVALID / INVALID + 理由コードを出力します。
- 処理後に判定結果の集計（SUMMARY）を表示します。
- 不正入力があっても途中で終了せず最後まで処理します。

## 学籍番号の仕様
本プログラムでは、学籍番号を以下の形式として扱います。
```
YYDDDNN

YY:入学年（20〜29）
DDD:学科コード（030 のみ有効）
NN:個別番号（01〜99）
```

## 使用方法

### インストール（クローン）
- 以下のコマンドを実行してください。
```
git clone https://github.com/rinron777/robosys2025-7.git

cd robosys2025-7
```

### 実行方法
- 標準入力から学籍番号を与えて実行します。
```
cat input.txt | ./count
```
- または直接入力することもできます。
```
./count


（Ctrl+D で入力終了）
```
### 実行例
```
$ cat <<EOF | python3 count.py
2303007
2310001
23A3001
2303999
2303008
EOF
```
### 出力例
```
2303007: VALID
2310001: INVALID E04_DEPT
23A3001: INVALID E02_NON_DIGIT
2303999: INVALID E04_DEPT
2303008: VALID

--- SUMMARY ---
VALID: 2
INVALID: 3
E01_LENGTH: 0
E02_NON_DIGIT: 1
E03_YEAR: 0
E04_DEPT: 2
E05_NUMBER: 0
```

## エラーコード一覧
```
エラーコード/内容
E01_LENGTH/桁数が不正
E02_NON_DIGIT/数字以外の文字が含まれている
E03_YEAR/年の範囲が不正
E04_DEPT/学科コードが不正
E05_NUMBER/個別番号が不正
```
